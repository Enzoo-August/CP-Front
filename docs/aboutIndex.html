Resumo do projeto

Esse trabalho foi construido com o intuito de desenvolver um web site para a empresa Upgrade. A UPGRADE é uma empresa que desde 2010 atua no segmento de produtos para segurança e acessibilidade, visando sempre oferecer produtos com alto padrão de qualidade e proporcionar a satisfação dos clientes através de soluções adequadas para as suas necessidades, atendimento e preço justo. Os produtos comercializados pela UPGRADE são fabricados por uma empresa do mesmo grupo, fabricante de peças para o segmento automotivo, com o Sistema de Gestão da Qualidade certificado pela Norma NBR ISO 9001. As barras de apoio da UPGRADE são fabricadas em conformidade com os requisitos da Norma NBR 9050, e são utilizados materiais comprados com certificado de qualidade dos fornecedores, sendo assegurada a sua qualidade e procedência. Os nossos produtos, além de auxiliar as pessoas idosas e portadoras de necessidades especiais a se movimentarem durante o banho ou no uso do vaso sanitário, também são destinados para as pessoas que desejam ter maior segurança, conforto e comodidade. Além dos produtos relacionados no site, também fabricamos peças com medidas especiais, conforme projeto ou medidas fornecidas pelo cliente. Este trabalho foi feito para diciplina FronEnd do professor Caio Oliveira.

Este projeto contém :

  • Readme
  • Página produtos, focada em ilustrar os diferentes tipos de produtos da empresa
  • Página empresa, contém toda a história e princípios da empresa
  • Página contatos, especializada para o cliente poder entrar em contato com a empresa
  • Página home, (htmlindex) será a página que vai abrir primeiro, e disponibilizara a navegação do site
